# Changelog: IntelliJ Line Debugging

All notable changes to this skill are documented here.

## [Unreleased]

### Changed
- No unreleased changes yet.

## [0.1.1] - 2026-03-19

### Changed
- Expanded the skill to cover editable local dependency stepping and local repo path disclosure during IntelliJ debugging
- Clarified that reusable IDE-facing debug helpers should live in `tutorials/` or another obvious human-facing debug folder
- Added examples and trigger prompts for tutorial-style debug helpers and editable dependency debugging

## [0.1.0] - 2026-03-18

### Added
- Initial version of `intellij-line-debugging`
- Guidance for IntelliJ-first, line-by-line debugging with serial real-input flows
- Examples covering debug helpers, direct entrypoint debugging, and third-party library stepping
- Test prompts for validating the skill trigger
