# Changelog

All notable changes to the Strategy Impact Log skill are documented here.

## [1.0.0] - 2026-04-02

### Initial Release

- Created comprehensive skill for tracking strategy proposals, code changes, and their effects
- Defined core principles for durable strategy recording
- Established step-by-step process for creating and updating strategy entries
- Documented common patterns (A/B comparison, iterative refinement, dead ends)
- Created detailed examples covering success, failure, comparison, iteration, and abandonment scenarios
- Added condensed checklist for quick reference
- Included rules for maintaining honest, traceable strategy records

### Features

- Strategy entry template with proposal, implementation, metrics, and learnings sections
- Integration points for commit hashes, file changes, and performance data
- Pattern library for common strategy decision scenarios
- Example galleries for multiple outcome types
- Guidance on using the log to inform future decisions

### Documentation

- INSTRUCTIONS.md: Detailed guide with principles and step-by-step process
- EXAMPLES.md: Five worked examples covering different scenario types
- SKILL.md: Exportable skill definition for use with Claude
